<h1>Sobre</h1>
<p>Décimo primeiro projeto do módulo 3 - Back-end, da <a href="https://betrybe.com" target="_blank">Trybe!</a></p>

<p>Tivemos 32 desafios para solucionar com as querys do MongoDB.</p>
